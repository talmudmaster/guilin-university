输出格式:

输入样例:

在这里给出一组输入。例如：

A man,a plan; cnalPanama

输出样例:

在这里给出相应的输出。例如：

yes

PYTHON3
代码
str = input()
str = str.replace(" ","")
str = str.lower()
list = []
for i in range(len(str)):
    if str[i].isalpha():
        list.append(str[i])
if list == list[::-1]:
    print("yes")
else:
    print("no")
