何时才能返校
(15分)

突如其来的新冠肺炎疫情阻碍了我们返校的脚步，我们已经多久没有看到美丽的校园了？我们何时才能返校啊？假设现在我们接到通知某日终于可以返校了，让我们一起来算一下我们离开校园的日子到底有多久。（假设寒假从2020年1月16日开始）

输入格式:

输入一行两个整数表示返校日的月和日。

输出格式:

输出从寒假开始到返校日我们离开校园的天数（返校当日不算在内）。

输入样例:

4 20

输出样例:

在这里给出相应的输出。例如：

95 days
编译器
PYTHON3
代码

import datetime

s = input().split(' ')
result = []
for i in s:
    if len(i) == 1:
        result.append(i.rjust(2, '0'))
    else:
        result.append(i)
d1 = datetime.datetime.strptime('-'.join(result), '%m-%d')
d2 = datetime.datetime.strptime('1-16', '%m-%d')
delta = d1 - d2
print(delta.days + 1, end=' ')
print('days')
